### Права доступа

Доступы – это первый уровень безопасности, предлагаемый архитектурой расширений. Они позволяют уменьшить ущерб от зараженных расширений или приложений. Например, если расширение, имеет доступ к манипуляции закладками, то оно не сможет влиять на загрузки пользователя так как закрыт доступ. Следовательно, при разработке расширения вы должны указать только нужные доступы используемых API.

Так же при установке определённых расширения будет показано предупреждение для получения больших прав доступа. Более подробно познакомиться с данными предупреждения можно по ссылке [https://developer.chrome.com/extensions/permission\_warnings](https://developer.chrome.com/extensions/permission_warnings).
